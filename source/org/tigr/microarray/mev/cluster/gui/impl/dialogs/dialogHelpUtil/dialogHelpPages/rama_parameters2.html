Running Rserve

The Rserve Daemon must be running for Rama to work. A RAMA user manual can be found at http://www.expression.washington.edu:8080/ramadoc/MeV-R_Documentation.html This manual details installation and configuration of R and Rserve. The manual can also be found in the Documentation section of this Software Package.

Rserve on Windows

Simply double click on Rserve.exe.

Rserve on OS X

Open a terminal window. Type: 'R CMD Rserve' and hit enter to start the daemon.
