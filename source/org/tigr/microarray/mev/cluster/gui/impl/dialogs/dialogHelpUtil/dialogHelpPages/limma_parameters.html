LIMMA: Linear Models for Microarray Data

Parameter Information

BETR is a flexible linear random-effects modeling framework that takes into account correlations between samples and sampling times.

Type of Data

Choose the type of data that BETR will run on.
2-Condition: Each condition must have multiple replicates for each time-point.
1-Condition: Only one condition, the initial time-point will be treated as the control.
Paired data: Data exists as a difference between two-conditions.

Choosing Time and Conditions

After choosing the type of data and number of time-points a dialog appears for time and condition assignments. You may assign your samples using either the individual button selection method or the cluster selection method.
For each condition every time-point must be assigned at least two samples.

Alpha value

This text field allows you to enter the alpha or critical p-value.

Hierarchical Clustering

This check box specifies whether to perform hierarchical clustering on the elements in each cluster created
