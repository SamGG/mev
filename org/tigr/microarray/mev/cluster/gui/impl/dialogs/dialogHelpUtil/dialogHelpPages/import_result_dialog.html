Import Result Report

Overview

This dialog reports the result of an identifier list import. The number of identifiers that had matching elements (gene or samples) in the data set, a list of those matching elements, a list of identifiers that were found, and a list of identifiers that were not found are all reported.

Genes or Samples Matched Panel

This top area of the window reports the number of identifiers matches and provides a table listing all matching elements. This section also provides button shortcuts to select all or clear the selected elements in the table. Note that the Store Cluster button will store the selected indices as a cluster.

Matching Results

Two lists are provided to account for all input identifiers. The list segregates the input identifiers into those that had matches in the data set and a separate list for those identifiers not found to match elements in the data set.

A Note on the Results

It can be the case that the resulting list size might not match the number of identifiers entered. It might be that some identifiers didn't match an element in the input data or it can be the case that there were multiple genes in the data set that matched a particular identifier.
