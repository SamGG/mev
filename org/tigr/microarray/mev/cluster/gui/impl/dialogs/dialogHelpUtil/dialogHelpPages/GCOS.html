Affymetrix GCOS File Format

A Affymetrix GCOS file is a tab-delimited text file that contains Affymetric Gene chip ID and several experiment datum. In each experiment data it contains one column of intensity, one column of detection call and one column of p-value. The first line is head line.
A sample data file is displayed as following.
GeneIDSample1-IntensitySample1-detection call(optional)Sample1-pvalue(optional)Sample2-IntensitySample2-detection call(optional)Sample2-pvalue(optional)
gene1 0.22 p 0.001 0.42 A 0.5
