Delta Dialog

Each Delta/Rho combination that was used during cross validation is listed here along with the # of genes used and the # of mistakes/# of classifications that resulted from using said Delta/Rho. It is up to the user to decide what Delta/Rho to be used for classification.

By default, the user will be prompted to save their 'Training Results'. The user may uncheck 'Save Training Results' if desired.
