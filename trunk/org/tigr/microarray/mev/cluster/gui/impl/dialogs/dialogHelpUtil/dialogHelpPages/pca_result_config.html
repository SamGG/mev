PCA 3D Result Configuration

Parameter Information

Axis Scaling Parameters

The first three parameters (Scale axix X, ...Y, and ...Z) are for scaling the X, Y, and Z axis ranges. The entered value is the viewable distance on either side of the origin.

Point Size

This floating point value scales the size of each element point in space.

Selected (Point Size)

This floating point value scales the size of each element point which has been selected.
