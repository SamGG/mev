Script Attribute Initialization

The Script Attribute Input Dialog is presented when creating a new script. These attributes can be used to identify the script by defining a descriptive name and a brief description of the analysis. These attributes are stored within the script as comments.

Creation Date

The script creation date it recorded by default during script creation.

Script Name

This optional attribute can be used as a short descriptive identifier.

Description

This optional attribute is used to document more detail about the script's algorithms and other information related algorithm choices and experimental factors.
